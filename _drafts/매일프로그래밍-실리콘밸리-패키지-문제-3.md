Given an integer N, find the number of possible balanced parentheses with N opening and closing brackets.



예제)

Input: 1

Output: ["()"]



Input: 2

Output: ["(())", "()()"]



Input: 3

Output: ["((()))", "(()())", "()(())", "(())()", "()()()"]
